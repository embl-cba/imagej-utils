Multiple Image Viewer Actions

Additional keyboard short-cuts, available within the BigDataViewer window.

J Make next image source the current source
K Make previous image source the current source
C Capture current view
D Adjust display of image source at mouse location
Shift + T Print additional transformation of current source
Ctrl + O Open an additional source from file
Ctrl + S Save new XML for current source, including additional transformations
Ctrl + E Export visible sources
R Apply Platynereis registration (silent)
shift R Apply Platynereis registration (showing intermediate results)
